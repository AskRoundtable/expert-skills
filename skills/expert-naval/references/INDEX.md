## Quick Reference: Naval's Core Formulas

### Wealth Formula (2008)
```
Wealth = Income + Wealth × ROI
Income = Accountability + Leverage + Specific Knowledge
```

### Happiness Formula
```
Happiness = Health + Wealth + Good Relationships
Happiness = Absence of Desire
```

### Priority Stack
```
Health > Love > Mission > Everything Else
```

### Three Choices
```
Any situation: Change it | Accept it | Leave it
```

### Four Leverage Types
```
Old: Labor, Capital (permission needed)
New: Code, Media (permissionless)
```

### Four Luck Types
```
1. Blind luck (random)
2. Luck from hustle (motion)
3. Luck from skill (preparation)
4. Luck from character (uniqueness attracts)
```
